StreamDock Features

Everything you need to manage your Icecast streaming server

Station Creation Wizard

Create new radio stations with a guided 3-step wizard. Auto-generated credentials, mount points, and one-click copy for all connection details.

Station Management

View all your stations at a glance with station cards showing server, port, mount point, and password. Toggle password visibility and delete with confirmation.

Live Status Detection

Real-time polling checks Icecast every 5 seconds. Animated LIVE badge and listener count. Visual highlights with green ring around active stations.

Built-in Audio Player

One-click playback of live streams. Play/pause controls toggle stream audio in-browser. Auto-stops when station goes offline.

Dashboard

Station overview with quick stats on active vs total stations. Server status indicator and quick start guide for new users.

Encoder Settings

Copy-ready connection details for BUTT, OBS, Mixxx, and other streaming software. MP3 or AAC format with Low/Standard/High quality presets.

Secure HTTPS Streaming

Stream securely via HTTPS proxy. Both secure and legacy (non-SSL) URLs provided. Compatible with StationDock stream monitoring.

Icecast Status Page

Secure proxy to Icecast status page for debugging. JSON API for programmatic access. Full Icecast admin interface available.

StationDock Integration New

Seamless integration with StationDock for stream monitoring and automated recording. Test streams directly from the monitoring dashboard.
