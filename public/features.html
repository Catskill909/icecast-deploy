Everything you need to manage your streams

StreamDock is a modern interface for Icecast streaming. Create stations, manage encoders, and monitor live streams with ease.

View App Screenshots

Station Creation Wizard

  • Create stations with a guided 3-step wizard
  • Auto-generated credentials and mount points
  • One-click copy for all connection details

Station Management

  • View all stations at a glance with cards
  • Show server, port, mount, and password
  • Delete stations with confirmation modal

Live Status Detection

  • Real-time polling every 5 seconds
  • Animated LIVE badge and listener count
  • Green ring highlight on active stations

Built-in Audio Player

  • One-click playback of live streams
  • Play/pause controls in browser
  • Auto-stops when station goes offline

Dashboard

  • Quick stats on active vs total stations
  • Server status indicator
  • Quick start guide for new users

Encoder Settings

  • Copy-ready details for BUTT, OBS, Mixxx
  • MP3 or AAC format options
  • Low/Standard/High quality presets

Secure HTTPS Streaming

  • Stream securely via HTTPS proxy
  • Legacy HTTP URLs still available
  • Compatible with StationDock monitoring

Icecast Status Page

  • Secure proxy to Icecast status page
  • JSON API for programmatic access
  • Full admin interface available

Real-Time Alerts

  • Notified when stream goes live or offline
  • Listener milestone alerts (50, 100, 250+)
  • Per-station email recipients
  • Smart routing with global fallback

StationDock Integration

  • Seamless stream monitoring integration
  • Automated recording support
  • Real-time health checks

Admin Authentication

  • Password-protected dashboard access
  • Secure HTTP-only session cookies
  • Easy setup via environment variable
  • Logout from Admin dropdown menu

Stream Relay & Restreaming New

  • Pull audio from external stream URLs
  • Primary mode (no encoder needed)
  • Fallback mode (auto-switch on disconnect)
  • URL validation with format detection
  • Supports MP3, AAC, Ogg, Opus
